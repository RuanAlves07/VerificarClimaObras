# Verificar Clima Obras

Um projeto para verificação e monitoramento de condições climáticas em obras.

## Descrição

Este projeto utiliza Python com a biblioteca **greenlet** para processamento assíncrono e eficiente de dados climáticos relacionados a obras em andamento. O greenlet permite a execução de múltiplas corrotinas de forma leve e performática.

## Requisitos

- Python 3.12+
- XAMPP (ambiente local)
- pip (gerenciador de pacotes Python)


## Estrutura do Projeto

```
VerificarClimaObras/
├── venv/                    # Ambiente virtual Python
├── src/                     # Código-fonte principal
├── requirements.txt         # Dependências do projeto
└── README.md               # Este arquivo
```

## Dependências Principais

- **greenlet 1.0.0+**: Implementação eficiente de corrotinas para processamento assíncrono
